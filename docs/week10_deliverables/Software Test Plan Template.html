1. Overview

In this project we use Jest to test our project. Jest is used by Facebook to test all JavaScript code including React applications. One of Jest's philosophies is to provide an integrated ?zero-configuration? experience.

1.1 Test Objectives

We use Jest to test whether is function as we want it to be. And have the react functions we expect.  

1.2 Test Environment

We use our local machine and running on windows 10, Linux redhat, and MacOS to test the code. Running our website on our school's server silo. Using ssh key to log into our silo account and set up the test environment. Networking between all the machines in the beginning is just running on our local machine.
We had our local database set up and connect the local database with the website. Later we have our server set up which is silo. We push the website to the server run it from there. Support software packages are node.js, bootstramp. react-redux framework. Django framework need to be installed. We have the test environment on our local machine for now, will transfer from local machine to server. With local machine we have more conveniency. With remote server it has the possibility crash.

1.3 Test Personnel

Scott and Yuanzhi are testing for the front end part. Both will responsible for creating react-redux front end test. Karanpaul and Shweta are testingh for the back end part. Both will responsible for creating user cases and data to test the website.
We will have our user involvement more and more along with our project developing.

1.4 Acceptance Criteria

WIth a fully functional website with all the important features be correctly implemented such as Login, User home page, topic page, search for users, topics, private chat feature, and with the percentage of less than 1% errors, We will consider our website could be deliverable.

1.5 Noted Omissions

Whether the user home page is structured as we wanted should not verified by this test. The reason is that during this sprint we only need the basic three pages to be connected. And have the basic structure for it.
